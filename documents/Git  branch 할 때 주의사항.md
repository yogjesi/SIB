## branch 이름

- 이름의 중복을 피하기 위해 브랜치 이름에 규칙성 부여

  날짜 + 이름 + 번호

  ex) 1217 jiwon 01

  하루에 두 번 이상 커밋할 경우

  ex)1217 jiwon 02
